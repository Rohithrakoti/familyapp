# Mana Kutumbam

Welcome to the "Mana Kutumbam" family website project! This project serves as a digital space for our family to connect, share events, and showcase memories.

## Project Structure

```
Mana Kutumbam
├── index.html        # Home page of the family website
├── events.html       # Page displaying family events
├── gallery.html      # Page showcasing family photos and videos
├── social.html       # Page linking to family social media accounts
├── css
│   └── styles.css    # Styles for the website
├── js
│   └── script.js     # JavaScript functionality for the website
└── README.md         # Documentation for the project
```

## Features

- **Home Page**: The main landing page with links to other sections of the website.
- **Events Page**: A dedicated page for listing and adding family events.
- **Gallery Page**: A visual showcase of family memories through photos and videos.
- **Social Media Page**: Links to our family's social media accounts and feeds.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the home page.
3. Navigate through the links to explore other pages.
